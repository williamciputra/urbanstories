# Urbanstories Launch Checklist

Version: 1.0

Status: Draft

---

# Purpose

Dokumen ini berisi daftar pemeriksaan yang harus diselesaikan sebelum Urbanstories diluncurkan ke publik.

Seluruh item harus diperiksa kembali meskipun sebelumnya telah diuji selama proses pengembangan.

---

# 1. Infrastructure

* [ ] Domain aktif
* [ ] DNS telah dikonfigurasi
* [ ] SSL aktif
* [ ] HTTPS aktif
* [ ] Redirect HTTP → HTTPS
* [ ] Redirect www → non-www

---

# 2. Frontend

* [ ] Homepage berjalan normal
* [ ] Navigation berfungsi
* [ ] Search berfungsi
* [ ] Category Page berfungsi
* [ ] Author Page berfungsi
* [ ] Article Page berfungsi
* [ ] Related Articles tampil
* [ ] Mobile responsive
* [ ] Tablet responsive
* [ ] Desktop responsive

---

# 3. CMS

* [ ] WordPress berhasil diinstal
* [ ] Login administrator
* [ ] Permalink dikonfigurasi
* [ ] Timezone benar
* [ ] Category dibuat
* [ ] User administrator dibuat
* [ ] REST API aktif

---

# 4. Editorial

* [ ] Editorial Guidelines disetujui
* [ ] Dateline diterapkan
* [ ] Featured image menggunakan rasio 16:9
* [ ] Slug sesuai standar
* [ ] Excerpt diisi
* [ ] Meta Title diisi
* [ ] Meta Description diisi

---

# 5. SEO

* [ ] robots.txt tersedia
* [ ] sitemap.xml tersedia
* [ ] Metadata tampil
* [ ] Open Graph tampil
* [ ] Twitter Card tampil
* [ ] Canonical URL benar

---

# 6. Analytics

* [ ] Google Search Console
* [ ] Google Analytics 4
* [ ] Sitemap dikirim ke Search Console

---

# 7. Security

* [ ] Password administrator kuat
* [ ] Login administrator dibatasi
* [ ] Backup aktif
* [ ] WordPress terbaru
* [ ] Plugin terbaru

---

# 8. Performance

* [ ] Image optimization
* [ ] Core Web Vitals diperiksa
* [ ] Tidak ada broken image
* [ ] Tidak ada broken link

---

# 9. Content

* [ ] Minimal 20 artikel siap tayang
* [ ] Seluruh artikel telah melalui proofreading
* [ ] Internal linking diterapkan
* [ ] Semua artikel memiliki featured image

---

# 10. Final Verification

* [ ] Semua halaman dapat diakses
* [ ] Tidak ada error di browser console
* [ ] Tidak ada error build
* [ ] Tidak ada dead link
* [ ] Tidak ada halaman placeholder
* [ ] Favicon tampil
* [ ] Manifest aktif
